# kse_sociology

`src/` - a js app that fetches data
`exploration/` -- jupyter notebooks where the research is done

## Data fetching

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run src/index.ts
```

## Research

```bash
cd exploration
python -m venv venv
source ./venv/bin/activate
pip install --upgrade -r requirements.txt
jupyter notebook
```
